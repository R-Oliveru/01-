# 01计划 - 产品孵化平台

## 项目概述
"01计划"是一个帮助个人或小团队持续打造100个产品的孵化平台。平台将产品创造过程分为三个阶段：
1. **需求收集** - 点子池、AI评审、人工验证
2. **需求开发** - 项目管理、进度跟踪
3. **GTM** - 产品上线、增长跟踪、盈利记录

## 开发理念
- **快速迭代**：采用 vibe coding 风格，快速原型验证
- **分阶段推进**：每个阶段确认后再进入下一阶段
- **AI增强**：利用AI进行点子评审、开发辅助

## 当前状态
- ✅ **第一阶段（需求收集）**：需求已确认，等待开发
- ✅ **第二阶段（需求开发）**：方案已确认，等待开发
- ✅ **第三阶段（GTM）**：**所有细节已确认**（看板视图、增长模板、数据录入方式）
- **开发状态**：**方案完全成熟，可以开始开发** 🚀
- **页面集成**：三个阶段集成在单一页面，通过顶部导航切换
- **引导页面**：首页显示项目统计（各阶段项目数量、累计用户数）

## GTM核心决策确认
- **累计用户数**：指产品的累计用户数（非平台用户）
- **主要目标**：记录历史数据、建立可复用的增长方法论流程
- **关键指标**：用户增长（日活、留存）、盈利情况（收入、成本、ROI）
- **页面视图**：看板视图（按状态分组：准备中/已上线/归档）
- **产品详情页**：基础信息、市场反馈、用户数据、财务数据
- **增长模板**：小红书冷启动、Geo定位推广、内网社区推广（固定+可自定义）
- **数据录入**：按需记录、不支持批量导入、无数据验证

## 整体架构
- **单页应用**：React + Vite + TypeScript
- **导航结构**：顶部固定导航栏 `[📝 需求收集] [🛠️ 需求开发] [🚀 GTM]`
- **数据流**：中央数据存储（IndexedDB），三阶段数据实时同步
- **默认视图**：打开应用时显示**引导页面**（项目统计概览）
- **引导页面内容**：
  - 📊 项目分布统计：各阶段项目数量
  - 👥 累计用户数（如果项目有用户数据）
  - 🎯 近期目标提醒
  - 🔄 快速入口到各阶段

## 已确认的技术决策
- **前端**：React + Vite + TypeScript
- **AI模型**：DeepSeek API（性价比高，中文优化好）
- **数据存储**：纯前端存储（IndexedDB），单机使用
- **用户模式**：单用户，无需登录
- **分类体系**：App需求/网页需求/小程序需求/Agent/工具库/内容产品/硬件产品/其他 + 自定义标签
- **AI评审权重**：用户价值 > 可行性 > 商业价值 > 趣味性
- **项目分解**：粗粒度（3-5个阶段）
- **进度跟踪**：基于任务完成数量自动计算
- **技术栈管理**：智能推荐 + 模板化选择
- **阶段转换**：一键从需求转换为开发项目
